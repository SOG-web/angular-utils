Getting Started

Installation

npm install @sog-web/angular-utils-font
# or
pnpm add @sog-web/angular-utils-font

Quick Example

import { inter } from '@sog-web/angular-utils-font/google';

export const myFont = inter({
  weights: [400, 500, 600],
  subsets: ['latin'],
  display: 'swap',
  variable: '--font-inter',
});
